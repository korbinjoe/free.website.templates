## üé® Customization

### Colors
The template uses an orange/red color scheme which can be customized in `tailwind.config.js`:

```javascript
colors: {
  primary: {
    50: '#fef7ee',
    500: '#f16725',
    600: '#e24e1b',
    // ... full color palette
  }
}
```

### Content
- Update company name and branding in `Header.tsx` and `Footer.tsx`
- Modify hero content in `Hero.tsx`
- Customize features in `FeaturesSection.tsx`
- Update API examples in `APISection.tsx`
- Edit testimonials in `TestimonialsSection.tsx`

### Styling
- All styles use TailwindCSS utilities
- Custom gradient text effect with `.gradient-text` class
- Responsive design with mobile-first approach
- Hover effects and transitions throughout
